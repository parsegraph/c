POST parsegraph.com/create-environment

Creates a new environment owned by the authenticated user.

Response

Content-Type: text/html

If the environment was successfully created, redirect 303 See Other with the Location: response header set to a URL of the created environment.

If the environment failed to create, respond 400 Bad Request with the user-readable status message.

Content-Type: application/json

Respond 200 OK if the environment was created, including a JSON object with the following fields. If the environment failed to create, then 400 Bad Request is given, with the JSON object still returned.

FieldValue
success

true if successfully created a environment, false if not.

status

A user-readable string that describes the status of the user's request, or the error that prevented its fulfillment.

guid

A 128-bit base-64 encoded string representing the GUID of the created environment.

This value is unspecified if success is false.
