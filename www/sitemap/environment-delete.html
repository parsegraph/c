DELETE parsegraph.com/<GUID>

Deletes the specified environment.

Response

Content-Type: text/html

If the environment was deleted, respond 200 OK with a indicative message.

If the environment failed to delete, respond 400 Bad Request with the user-readable status message.

Content-Type: application/json

Respond 200 OK if the environment was created, including a JSON object with the following fields. If the environment failed to create, then 400 Bad Request is given, with the JSON object still returned.

Field Value
success

true if successfully created a environment, false if not.

status

A user-readable string that describes the status of the user's request, or the error that prevented its fulfillment.
