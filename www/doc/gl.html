Parsegraph GL

GL functions.

parsegraph_generateRectangleVertices(x, y, w, h)

Returns a list of 2-D vertex coordinates that will create a rectangle, centered at the specified position.

getVerts(width, length, height)

Returns a [vec, vec, vec, ...] array of 24 vertices representing an origin-centered cube of the given size.

parsegraph_generateRectangleTexcoords()

Returns a list of texture coordinates for use with a face of a cube.

compileShader(gl, shaderSource, shaderType)

Creates and compiles a shader of the given type, returning the ID of the created shader.

Method from webglfundamentals.org.

createProgram(gl, vertexShader, fragmentShader)

Creates a program from 2 shaders, returning the ID of the linked program.

Method from webglfundamentals.org.

resize(gl)

Sets the GL viewport to the size of its owning canvas.

Method from webglfundamentals.org.

matrixIdentity3x3()

Returns a 3x3 identity matrix.

matrixCopy3x3(src)

Returns a new 3x3 matrix that is the row-major copy of the given matrix.

matrixMultiply3x3(...)

Multiplies all of the given 3x3 row-major matrices, in order, returning a new matrix equal to the result.

matrixTransform2D(world, x, y)

Returns the given point, transformed by the given world matrix.

makeTranslation3x3(tx, ty)

Returns a new 3x3 translation-matrix using the given values.

makeRotation3x3(angleInRadians)

Returns a new 3x3 rotation matrix, using the given angle.

makeScale3x3(sv)

Calls makeScale3x3(sv, sv).

makeScale3x3(sx, sy)

Returns a new 3x3 matrix of the given scaled dimensions.

makeInverse3x3(input)

Returns a new 3x3 matrix that is the inverse of the given matrix.

midPoint(x1, y1, x2, y2)

Returns a new 2D vector positioned between the two specified points.

make2DProjection(width, height)

Creates a 2D orthographic projection matrix using the specified viewport dimensions.

subtractVectors3D(a, b)

Returns a new vector that is equal to a - b.

normalize3D(v)

Returns a new 3-dimensional vector equal to the normalized value of the given 3D vector.

cross3D(a, b)

Returns a new 3D vector equal to the cross product of a and b.

makePerspective(fieldOfViewInRadians, aspect, near, far)

Creates a new 4x4 perspective matrix specified by the given values.

makeTranslation4x4(tx, ty, tz)

Returns a new 4x4 translation matrix.

makeXRotation(angleInRadians)

Returns a new 4x4 rotation matrix, rotated the given radians amount about the X axis.

makeYRotation(angleInRadians)

Returns a new 4x4 rotation matrix, rotated the given radians amount about the Y axis.

makeZRotation(angleInRadians)

Returns a new 4x4 rotation matrix, rotated the given radians amount about the Z axis.

makeScale4x4(sx, sy, sz)

Returns a new 4x4 scale matrix.

matrixMultiply4x4(a, b)

Returns a new 4x4 matrix that is the product of the two given 4x4 matrices.

makeInverse4x4(m)

Returns a new 4x4 matrix that is the inverse of the given matrix.

matrixVectorMultiply4x4(v, m)

Returns a new 4D vector equal to the given 4D vector multiplied by the given 4x4 matrix.

makeLookAt(cameraPosition, target, up)

Returns a 4x4 matrix that, positioned from the camera position, looks at the target, a position in 3-space, angled using the up vector.
