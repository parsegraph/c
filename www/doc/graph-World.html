Parsegraph World

A World object contains a number of rendered graphs.

parsegraph_World(graph)

Creates a new World object using the given graph.

.camera()

Returns the Camera used for this World, creating it if it does not yet exist.

.setCamera(camera)

Directly sets the Camera used for this World.

.plot(node, worldX, worldY, userScale)

Plots the given Node at the given coordinates.
  • If the given object is actually a Caret, then its root node is used.
  • Throws if the given object is null.
  • If the world position is not given, (0, 0) is assumed.

The paint group is returned. Passing a paint group directly is the preferred method. The position of the graph is the one used when rendering. The caller manages the position.

.removePlot(plot)

Removes the given plotted Node from this World.

.mouseOver(x, y)

Receives a mouseover event at the given coordinates, in client space.

Returns true if this event processing requires a graph repaint.

.boundingRect(outRect)

Returns a new parsegraph_Rect (or sets the values of the given Rect) to the bounding rectangle for all the plotted graphs in this World.

.scheduleRepaint()

Marks this World as dirty.

.nodeUnderCursor()

Directly returns the last node under the cursor.

.nodeUnderCoords(x, y)

Tests whether the given position, in world space, is within a node.

.needsRepaint()

Returns true if this World needs paint() to be called.

.paint(timeout)

Paints for the given time interval, in milliseconds.

If false is returned, painting did not complete.

.render(world)

Renders the painted scene to the GL context.
  • Throws if any root node does not have a paint group.
