parsegraph_NULL_STATUS

Constant that indicates a null Status.

parsegraph_OK

Constant that indicates success.

parsegraph_BAD_STATUS

Constant that indicates an unrecognized Status value.

parsegraph_NO_NODE_FOUND

Constant that indicates the node could not be found.

parsegraph_ALREADY_OCCUPIED

Constant that indicates the specified direction already has a node.

parsegraph_BAD_NODE_DIRECTION

Constant that indicates an unrecognized NodeDirection value.

parsegraph_BAD_NODE_CONTENT

Constant that indicates that the given content is invalid for this Node.

parsegraph_BAD_AXIS

Constant that indicates an unrecognized NodeAxis value.

parsegraph_BAD_LAYOUT_STATE

Constant that indicates an unrecognized LayoutStatus value.

parsegraph_BAD_NODE_ALIGNMENT

Constant that indicates an unrecognized NodeAlignment value.

parsegraph_CANNOT_AFFECT_PARENT

Constant that indicates the intended operation cannot affect this Node's parent.

parsegraph_OFFSET_IS_NEGATIVE

Constant that indicates that negative offsets are not allowed.

parsegraph_NODE_IS_ROOT

Constant that indicates that the given Node is a root and does not support the intended operation.

parsegraph_BAD_LAYOUT_PREFERENCE

Constant that indicates that the given Node is a root and does not support the intended operation.

parsegraph_nameStatus(given)

Returns a programmer-friendly string that describes the given Status value.
