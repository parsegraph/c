new parsegraph_TexturePainter(gl, textureId, texWidth, texHeight, shaders)

Returns a new painter using the given GL context, specified GL texture, and compiled shader map.

.texture

Returns this painter's given texture ID.

.drawWholeTexture(x, y, width, height, scale)

Draws the entire texture at the specified rectangle and given scale.

.drawTexture(iconX, iconY, iconWidth, iconHeight, x, y, width, height, scale)

Draws a portion of this painter's texture at the specified rectangle and given scale.

.clear()

Clears this painter's buffers.

.render(world)

Renders all painted images using the given matrix.
