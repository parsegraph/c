Parsegraph Color

Alpha-containing Color object. Component values are between 0 and 1, inclusive.

new parsegraph_Color(r, g, b)

Creates a new Color object, with the specified values, clamped between 0 and 1, inclusive.

The alpha is assumed to be 1.

new parsegraph_Color(r, g, b, a)

Creates a new Color object with the specified values, clamped between 0 and 1, inclusive.

.r()

Returns the red component of this color, a value between 0 and 1, inclusive.

.g()

Returns the green component of this color, a value between 0 and 1, inclusive.

.b()

Returns the blue component of this color, a value between 0 and 1, inclusive.

.a()

Returns the alpha component of this color, a value between 0 and 1, inclusive.

.setA(value)

Sets the alpha component of this color to the given value. The value is clamped to 0 and 1, inclusive.

.setR(value)

Sets the alpha component of this color to the given value. The value is clamped to 0 and 1, inclusive.

.setG(value)

Sets the green component of this color to the given value. The value is clamped to 0 and 1, inclusive.

.setB(value)

Sets the blue component of this color to the given value. The value is clamped to 0 and 1, inclusive.

.multiply(other)

Creates and returns a new color equal to each component of this color, multiplied by its complement component in the given color.

.premultiply(other)

Creates and returns a new Color that is this color, alpha-blended with the given color. If this color is opaque, then the returned color is equal to this color.

.asRGB()

Returns a string of the form "rgb(255, 255, 255)" for white and "rgb(0, 0, 0)" for black.

parsegraph_fromRGB(rgb, defaultAlpha)

Extracts a color from an RGB string, formatted like asRGB() would return. If the RGB string lacks an alpha value, defaultAlpha is used.

parsegraph_fromRGB(rgb)

Same as parsegraph_fromRGB(rgb, 255).

.clone()

Creates a new Color equal to this color.

.equals(others)

Returns true if all components of this color are equal to their complement component in the given color.

.interpolate(other, lerp)

Creates and returns a new Color that is pleasantly interpolated, as determined by the value lerp, between this color and the given color. lerp is a clamped value between 0 and 1, inclusive.

parsegraph_createColor(r, g, b, a)

Alias for new parsegraph_Color(r, g, b, a).
