Parsegraph Input

parsegraph_RESET_CAMERA_KEY

The name of the key used to reset the camera to its default position. Defaults to Escape.

parsegraph_CLICK_KEY

The name of the key used to simluate mouse clicks.

parsegraph_MOVE_UPWARD_KEY

The name of the key used to move the camera up.

parsegraph_MOVE_DOWNWARD_KEY

The name of the key used to move the camera down.

parsegraph_MOVE_BACKWARD_KEY

The name of the key used to move the camera LTR backward.

parsegraph_MOVE_FORWARD_KEY

The name of the key used to move the camera LTR forward.

parsegraph_CARET_COLOR

The color of the caret.

parsegraph_FOCUSED_SPOTLIGHT_COLOR

The color of the spotlight on the focused node.

parsegraph_FOCUSED_SPOTLIGHT_SCALE

The relative size of the focused node spotlight.

parsegraph_ZOOM_IN_KEY

The name of the key used to zoom the camera in.

parsegraph_ZOOM_OUT_KEY

The name of the key used to zoom the camera out.

new parsegraph_Input(graph, camera)

Constructs and attaches this input to the given graph's canvas.

.lastMouseCoords()

Returns the most recently detected mouse coords.

.lastMouseX()

Returns the most recently detected mouse X-coordinate.

.lastMouseY()

Returns the most recently detected mouse X-coordinate.

.SetListener(listener, thisArg)

Sets the input listener to the given value, or clears it if listener is null.

.UpdateRepeatedly()

Returns whether the current input state expects the scene to be re-rendered on animation timer, rather than only when scheduled through input.

.Update(t)

Updates the scene using the given elasped time and current input state.

Returns true if the input state modified the scene such that it will require rendering.

.Get(key)

Returns whether the specified key is considered pressed.

.Elapsed(key, t)

Returns the amount of time, in seconds, that the current key has been pressed, and resets the key's elapsed timer.

.paint()

Paints this scene's focusing spotlight and text caret.

.focusedNode()

Returns the node currently under focus.

.setFocusedNode(focusedNode)

Directly sets the node under focus.

.focusedLabel()

Returns the label currently under focus.

.render(world)

Renders this input's painted scene using the specified world matrix.

.Dispatch(...)

Calls this input's listener with the given arguments.
