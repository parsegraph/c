Parsegraph CameraBox

new parsegraph_CameraBox(graph)

Creates a new CameraBox component for the given Graph.

.needsRepaint()

Returns true if this component needs repainting.

.gl()

Returns the GL rendering context used by this component.

.glyphAtlas()

Returns the glyph atlas used by this component.

.prepare(gl, glyphAtlas, shaders)

Prepares this component for rendering with the given GL rendering arguments.

.setCamera(name, camera)

Sets the specified camera box to the given camera object with the following fields:
Field name
scale
width
height

.removeCamera(name)

Removes the specified camera box.

.paint()

Paints all camera boxes.

.render(world)

Renders all painted camera boxes using the given matrix
