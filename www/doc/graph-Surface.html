Parsegraph Surface

Manages a div containing a canvas rendered using the given listeners.

new parsegraph_Surface()

Creates a new Surface and its HTML elements. The GL context is not created until used.

.canvas()

Returns the canvas HTML element used for rendering this Surface.

.gl()

Returns the GL context used by this Surface, creating it if necessary.

.setGL(gl)

Directly sets the GL context used by this Surface.

.setAudio(audio)

Directly sets the audio context to the given value.

.audio()

Returns the AudioContext used by this Surface, creating one if necessary.
  • Throws an Error if AudioContext is not supported.

.resize(w, h)

Resizes this Surface's container to the specified size, in pixels.

.container()

Returns the HTML container element that holds the canvas for this graph.

.addPainter(painter, thisArg)

Adds a new painter function and this-object for this Surface.

.addRenderer(renderer, thisArg)

Adds a new renderer function and this-object for this Surface.

.paint(...)

Invokes all painter functions with the given arguments.

.setBackground(color)

Sets this Surface's background color to the given Color value.

.setBackground(...)

Sets this Surface's background color to the specified color.

.backgroundColor()

Retrieves the current background color.

.canProject()

Returns whether the surface has a nonzero client width and height.

.render()

Invokes all renderers.
  • Throws an Error if canProject() returns false.
