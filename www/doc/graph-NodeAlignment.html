Parsegraph NodeAlignment

Alignment constants used to position child nodes.

parsegraph_NULL_NODE_ALIGNMENT

Constant that refers to a null NodeAlignment value.

parsegraph_DO_NOT_ALIGN

Constant that indicates the node in the given direction should be aligned without consideration to its descendants.

parsegraph_ALIGN_NEGATIVE

Constant that indicates the node in the given direction and its descendents should be aligned in the negative direction relative to its parent.

parsegraph_ALIGN_CENTER

Constant that indicates the node in the given direction and its descendents should be centrally aligned relative to its parent.

parsegraph_ALIGN_POSITIVE

Constant that indicates the node in the given direction and its descendents should be aligned in the positive direction relative to its parent.

parsegraph_ALIGN_HORIZONTAL

Constant that indicates the given inward Node should be aligned horizontally relative to its container's label.

parsegraph_ALIGN_VERTICAL

Constant that indicates the given inward Node should be aligned vertically relative to its container's label.

parsegraph_nameNodeAlignment(given)

Returns a programmer-friendly string that describes the given NodeAlignment value.
  • Throws a new parsegraph_BAD_NODE_ALIGNMENT error if the given value is unrecognized.
