parsegraph_Line

new parsegraph_Line(label, text)

Constructs a new glyph-run using the given text.

.isEmpty()

Returns whether the length of this line is zero.

.glyphAtlas()

Returns this line's label's glyph atlas.

.remove(pos, count)

Removes code characters from the zero-indexed given glyph position.

.appendText(text)

Appends the given text to this line.

.insertText(pos, text)

Inserts the given text into this line at the specified zero-index glyph position.

.getText()

Returns a string representing the text contained by this line.

.text()

Alias for .getText().

.linePos()

Directly returns this line's position.

.label()

Directly returns this line's owning label.

.width()

Directly returns this line's computed width.

.height()

Directly returns this line's computed width.

.posAt(limit)

Returns the horizontal offset of the glyph at the given zero-indexed position.

.glyphs()

Directly returns the glyph array used by this line.

parsegraph_Label

new parsegraph_Label(glyphAtlas)

Constructs a new unwrapped, non-editable label using the given glyph atlas.

Throws an error if no glyphAtlas is given.

.glyphAtlas()

Returns the glyph atlas used by this label.

.isEmpty()

Returns true if all lines of this label are empty, or if this label contains no lines.

.forEach(func, funcThisArg)

Calls func for each line of this label.

.getText()

Returns the text of all lines of this label, concatenated using newlines.

.text()

Alias for .getText().

.setText(text)

Sets this label's lines to the given text. Any existing text is removed.

.moveCaretDown(world)

NYI

.moveCaretUp(world)

NYI

.moveCaretBackward(world)

Moves the caret backward in the current line, or shifts it to the previous line if needed.

.moveCaretForward(world)

Moves the caret forward in the current line, or shifts it to the next line if needed.

.backspaceCaret()

Removes the character before the caret's position in the current line, shifting to the previous line if needed.

.deleteCaret()

Removes the character after the caret's position in the current line, shifting the next line if needed.

.ctrlKey(key)

Handles a control-key-pressed event of the given name.

.key(key)

Handles a key-pressed event of the given name.

By default, a printable character is inserted into the current line at the caret's position.

.onTextChanged(listener, listenerThisArg)

Sets this label's listener for events that change the label's text.

.textChanged()

Invokes this label's listener.

.editable()

Returns whether this label is editable using user events.

.setEditable(editable)

Sets whether this label is editable using user events.

.click(x, y)

Handles a mouse-clicked event using the specified coordinates, local to this label's origin.

.lineAt(n)

Returns the line at the given zero-indexed offset.

.caretLine()

Returns the line index of this label's caret.

.caretPos()

Returns the character index of this label's caret in the current line.

.getCaretRect(outRect)

Returns a rectangle (or uses outRect if given) covering the caret's current glyph position

.glyphPos()

Alias for .caretPos().

.fontSize()

Returns this label's glyph atlas's reported font size.

.glyphAtlas()

Returns this label's glyph atlas.

.width()

Returns the minimum width that will cover all lines of this label.

.height()

Directly returns the height of this label.

.paint(painter, worldX, worldY, fontScale)

Paints this label's text using the given glyph painter and world coordinates.
