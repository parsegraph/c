Parsegraph NodeDirection

parsegraph_readNodeDirection(given)

Returns a constant that represents the given named node direction.

parsegraph_nameNodeDirection(given)

Returns a string that names the given node direction constant.

parsegraph_isNodeDirection(given)

Alias for parsegraph_nameNodeDirection(given).

parsegraph_reverseNodeDirection(given)

Reverses the given node direction.

parsegraph_turnLeft(given)

Returns the node direction that is a turn to the left of the given node direction.
  • Throws an error if the given direction is not a valid direction.

parsegraph_turnRight(given)

Returns the node direction that is a turn to the right of the given node direction.
  • Throws an error if the given direction is not a valid direction.

parsegraph_turnPositive(direction)

Returns the node direction that is a turn in the positive node direction.

parsegraph_turnNegative(direction)

Returns the node direction that is a turn in the negative node direction.

parsegraph_isCardinalDirection(given)

Returns true if the given node direction is either parsegraph_UPWARD, parsegraph_DOWNWARD, parsegraph_FORWARD, or parsegraph_BACKWARD.
  • Throws an error if the given direction is not a valid direction.

parsegraph_nameAxis(given)

Names the given Node axis.

parsegraph_getNodeDirectionAxis(given)

Returns the node axis for the given Node direction.

parsegraph_isVerticalNodeDirection(given)

Returns true if the given direction is along the vertical Node axis.

parsegraph_isHorizontalNodeDirection(given)

Returns true if the given direction is along the horizontal Node axis.

parsegraph_getPerpendicularAxis(axisOrDirection)

Returns the perpendicular axis for the given Node direction or Node axis.

parsegraph_getPositiveNodeDirection(given)

Returns the positive Node direction for the given Node axis.

parsegraph_getNegativeNodeDirection(given)

Returns the negative Node direction for the given Node axis.

parsegraph_forEachCardinalNodeDirection(func, thisArg)

Calls the given function for each Node direction.

parsegraph_isPositiveNodeDirection(given)

Returns true if the given Node direction is positive.

parsegraph_isNegativeNodeDirection(given)

Returns true if the given Node direction is negative.

parsegraph_nodeDirectionSign(given)

Returns 1 if the given Node direction is positive, -1 otherwise.

parsegraph_alternateNodeDirection(given)

Returns an alternating series of Node directions, using the given Node direction as a base.
