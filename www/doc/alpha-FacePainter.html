Alpha FacePainter

Draws 3d faces in a solid color.

new alpha_FacePainter(gl)

Returns a new painter using the given GL context.
  • Throws an Error if the painter is not given a GL context.

.Clear()

Clears this painter's buffers.

.Quad(v1, v2, v3, v4, c1, c2, c3, c4)

Draws a quad with the given vertices and respective Colors.

.Triangle(v1, v2, v3, c1, c2, c3)

Draws a triangle with the given vertices and respective Colors.

.Draw(viewMatrix)

Draws the painted faces using the given matrix.

Alpha FacePainter

  1. new alpha_FacePainter(gl)
  2. .Clear()
  3. .Quad(v1, v2, v3, v4, c1, c2, c3, c4)
  4. .Triangle(v1, v2, v3, c1, c2, c3)
  5. .Draw(viewMatrix)
