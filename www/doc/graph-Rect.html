new parsegraph_Rect(x, y, width, height)

Creates a new rect centered at the specified position and of the specified size.

parsegraph_createRect(x, y, width, height)

Alias for parsegraph_Rect's constructor.

.x()

Returns this rect's X position.

.y()

Returns this rect's Y position.

.setX(x)

Sets this rect's X position to the given value.

.setY(y)

Sets this rect's Y position to the given value.

.height()

Returns this rectangle's height.

.setHeight(h)

Sets this rect's height to the given value.

.width()

Returns this rectangle's width.

.setWidth(h)

Sets this rect's width to the given value.

.toString()

Returns a developer-friendly string representing this rect.

.vMin()

Returns the minimum Y-coordinate of this rect.

.vMax()

Returns the maximum Y-coordinate of this rect.

.hMin()

Returns the minimum X-coordinate of this rect.

.hMax()

Returns the maximum X-coordinate of this rect.

.include(bx, by, bwidth, bheight)

Enlarges this rect as necessary to contain the specified rect.
