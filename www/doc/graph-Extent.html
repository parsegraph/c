Parsegraph Extent

Represents a list of bounds. Each bound is comprised of a length and a size. The extent always extends in the positive direction.

An extent is used to represent each forward, backward, upward, and downward side of a graph. The four extents represent the combine bounding picture of the graph.

parsegraph_DEFAULT_EXTENT_BOUNDS

Initial extent bound capacity.

new parsegraph_Extent()

Creates a new, empty bounds array.

new parsegraph_Extent(copy)

Copy another extent's array.

parsegraph_NUM_EXTENT_BOUND_COMPONENTS

The number of components per extent bound.

.forEach(func, thisArg)

Iterates over the extent, calling the given function for each bound. 
extent.forEach(function(length, size, i) {}, this);

.clone()

Returns a new extent that is a copy of this one.

.clear()

Sets the size of this extent to zero.

.numBounds()

Returns the number of bounds in this extent.

.hasBounds()

Returns whehter this extent's size is positive and non-zero.

.boundLengthAt(index)

Returns the length of the specified bound.

.boundSizeAt(index)

Returns the size of the specified bound.

.setBoundLengthAt(index, length)

Sets the length of the specified bound to the given value.

.setBoundSizeAt(index, size)

Sets the size of the specified bound to the given value.

.realloc(capacity)

Reallocates the extent to the new capacity.

Throws an error if the given capacity is smaller than the current capacity.

.prependLS(length, size)

Prepends a new bound of the given dimensions to this extent.

Throws an error if length is NaN. NaN sizes represent empty spaces.

.boundCapacity()

Returns this extent's capacity. Guaranteed to be >= .numBounds().

.appendLS(length, size)

Appends a new bound of the given dimensions to this extent.

.prependSL(size, length)

Prepends a new bound of the given dimensions to this extent.

.appendSL(size, length)

Appends a new bound of the given dimensions to this extent.

.adjustSize(adjustment)

Adds the given size adjustment to all bounds of this extent.

.simplify()

Converts this extent into a single bound that is the rectangular union of all bounds of this extent.

.sizeAt(offset)

Returns the size at the given positional offset.

.combineBound(newBoundStart, newBoundLength, newBoundSize)

Combines the defined bound into this extent.

.copyFrom(from)

Moves the bounds from the given extent into this extent.

.combineExtent(given, lengthAdjustment, sizeAdjustment, scale)

Combine the given extent into this extent, such that this extent exactly covers both its original bounds and the bounds of the given extent. The given extent can be transformed using lengthAdjustment, sizeAdjustment, and scale.

.combinedExtent(given, lengthAdjustment, sizeAdjustment, scale)

Returns a new Extent that exactly covers both this extent and the given extent, transformed by the given parameters.

.scale(factor)

Scales all bound's size and length by the given factor.

.separation(given, positionAdjustment, allowAxisOverlap, givenScale, axisMinimum)

Returns the minimum value that guarantees no overlap between the two extents.

Positive position adjustment requires this bound to be adjusted: 

 (this)
 <--*--->
    |
    <--|------->
    (given)

Negative position adjustment requires the given bound to be adjusted: 

  (given)
     <-------|-->
             |
          <--*-->
     (this)
The position adjustment is in this extent's space. So if the given scale is 0.5, the given extent's length and size are adjusted by that amount.

.boundingValues(outVal)

Computes and returns [totalLength, minSize, maxSize].

.equals(other)

Returns true if all this extent is component-wise equal with the given bound.

.dump(message)

Logs each bound of this extent using parsegraph_log, using the given optional message as the title.

.toDom(message)

Creates a HTML table describing this Extent.

parsegraph_createExtent(copy)

Alias for new parsegraph_Extent(copy).

parsegraph_checkExtentsEqual(caret, direction, expected, resultDom)

Returns true if the caret's node in the given direction is equal to the given expected extent, false otherwise.

If false and resultDom is given, then HTML elements are appended describing the difference between the actual extent in the caret node's given direction and the expected extent.
