Parsegraph Date

parsegraph_datesEqual(a, b)

Returns true if the year, month, and date parts of the given Dates are equal.

parsegraph_dateGreater(a, b)

Returns true if the year, month, and date part of a comes past b. If true, b is in the future relative to a.

parsegraph_getListOfDays()

Returns a new array of day names.

parsegraph_getListOfMonths()

Returns a new array of month names.

parsegraph_getRuntimeInMillis()

Gets the number of milliseconds seconds since the file was executed.

parsegraph_getTimeInMillis()

Gets the current time, in milliseconds.

parsegraph_outputMonth(d, includeYear)

Returns a string describing the month of the given Date, including the year if specified.

parsegraph_outputDate(d, includeDate, includeTime, includeToday)

parsegraph_previousMonth(d)

Returns a new Date representing the previous computed month from the given Date.

parsegraph_nextMonth(d)

Returns a new Date representing the next computed month from the given Date.

parsegraph_previousDay(d)

Returns a new Date representing 24 hours in the past from the given Date.

parsegraph_nextDay(d)

Returns a new Date representing 24 hours in the future from the given Date.

parsegraph_previousWeek(d)

Returns a new Date representing seven days in the past from the given Date.

parsegraph_nextWeek(d)

Returns a new Date representing seven days in the future from the given Date.

parsegraph_getFirstDayOfWeek(d)

Returns a new Date representing the Sunday of the same week as the given Date.

getSunriseAndSunset(inputDatetime, signedLongitude, signedLatitude)

Given a Date and a polar coordinate Earth position, return [sunrise in minutes since midnight, sunset in minutes since midnight].

From itacanet.org.

NormalizeTo360(n)

Normalizes the given value to be between -360 and 360 degress.

MonthToYearOffset(d)

Returns the day offset of the month of the given Date. (A value between 0 and the total days in the year).
