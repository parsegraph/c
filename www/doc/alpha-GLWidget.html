Alpha GLWidget

Alpha's world object.

new alpha_GLWidget(surface)

Creates a new GLWidget using the given Surface, or creating a Surface if necessary.

.paint()

Computes the scene for rendering.

.Tick(elapsed)

Advances the scene the given amount of time, in milliseconds.

.setBackground(color)

Sets the background to the given Color.

.setBackground(...)

Sets the background to the Color constructed using the given arguments.

.scheduleRepaint()

Marks this GLWidget as dirty and schedules a surface repaint.

.backgroundColor()

Retrieves the current background color.

.Camera()

Returns the widget's camera.

.canvas()

Returns the canvas used for rendering.

.gl()

Returns the GL context used for rendering.

.surface()

Returns the surface used for rendering.

.container()

Returns the container for this scene.

.render()

Render painted memory buffers.
