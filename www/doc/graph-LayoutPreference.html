Parsegraph LayoutPreference

LayoutPreference determines the order in which Nodes are laid out.

parsegraph_NULL_LAYOUT_PREFERENCE

Constant that refers to a null layout preference.

parsegraph_PREFER_PARENT_AXIS

Constant that indicates this Node lays out the child node along its parent's axis first.

parsegraph_PREFER_PERPENDICULAR_AXIS

Constant that indicates this Node lays out the child nodes perpendicular to the parent axis first.

parsegraph_PREFER_HORIZONTAL_AXIS

Constant that indicates this Node lays out its horizontal child nodes first.

parsegraph_PREFER_VERTICAL_AXIS

Constant that indicates this Node lays out its vertical child nodes first.

parsegraph_nameLayoutPreference(given)

Returns a string that describes the given LayoutPreference constant.
  • Throws an error if the given layout preference constant is unrecognized.
