parsegraph_CameraProtocol

A protocol to speak Parsegraph's Camera protocol over a given WebSocket.

parsegraph_CameraProtocol(ws, camera, eventFunc, eventFuncThisArg)

Creates a new object to represent the given camera using the given opened WebSocket.

The given eventFunc function and this-object are invoked when a protocol message is received.

  • Throws if eventFunc is not defined.

.update()

Schedules an update to be sent for this protocol's camera.

.onopen()

Handles opening events from the WebSocket.

.onmessage(event)

Handles message events from this protocol's WebSocket.

.onclose()

Handles close events from this protocol's WebSocket.

.opened()

Returns true if this protocol is accepting input.
