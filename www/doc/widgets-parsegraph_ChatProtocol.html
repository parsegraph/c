parsegraph_ChatProtocol

An object to speak Parsegraph's Chat protocol over a given WebSocket.

parsegraph_ChatProtocol(ws, eventFunc, eventFuncThisArg)

Creates a new unopened protocol using the given opened WebSocket.

The given eventFunc function and this-object are invoked when a protocol message is received.

  • Throws if eventFunc is not defined.

.send(text)

Sends the given text using this protocol's WebSocket. The text is sent in a JSON object with the following fields:
NameDescription
dataThe given text value.

.onopen()

Handles opening events from the WebSocket.

.onmessage(event)

Handles message events from this protocol's WebSocket.

.onclose()

Handles close events from this protocol's WebSocket.

.opened()

Returns true if this protocol is accepting input.
