Parsegraph Carousel

Visual graph menu component.

new parsegraph_Carousel()

Creates a new, unprepared Carousel centered at the origin.

.needsRepaint()

Returns whether this Carousel needs paint() to be called before render().

.prepare(gl, glyphAtlas, shaders)

Sets up the Carousel for rendering with the given rendering arguments.

.gl()

Returns the prepared GL context used for rendering.

.moveCarousel(worldX, worldY)

Sets the origin of this Carousel to the specified position.

.setCarouselSize(size)

Directly sets the carousel's size to the given value.

.showCarousel()

Sets the Carousel to be shown.

.isCarouselShown()

Returns true if the Carousel is set to be shown.

.hideCarousel()

Sets the Carousel to be hidden.

.addToCarousel(node, callback, thisArg)

Adds the given node to the carousel.

.clearCarousel()

Removes all this Carousel's plots and callbacks.

.removeFromCarousel(node)

Removes the given node from the Carousel, returning it if found.

.clickCarousel(x, y, asDown)

Receives a click event on the carousel, in client coordinates.

.mouseOverCarousel(x, y)

Receives a mouseover event on the carousel, in client coordinates.

.arrangeCarousel()

Arranges each carousel caret in a spiral.

.scheduleCarouselRepaint()

Calls this Carousel's repaint scheduler and marks this Carousel as needing a repaint.

.paint()

Paints the plotted graphs and selection frame for this Carousel.

.render(world)

Renders this painted Carousel using the given world matrix.

.backgroundColor()

Returns the parsegraph_Color used as the background.

.setBackgroundColor(backgroundColor)

Changes the parsegraph_Color used as the background.

.glyphAtlas()

Returns the GlyphAtlas for this carousel.

.render(world)

Renders this painted Carousel using the given world matrix.

.setOnScheduleRepaint(func, thisArg)

Sets the hook to be called when the carousel needs to be repainted.
