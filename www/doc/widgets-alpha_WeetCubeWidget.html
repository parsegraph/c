Alpha WeetCubeWidget

new alpha_WeetCubeWidget(surface)

Creates a new widget using the given Surface, creating it if necessary.

.Tick(elapsed, frozen, updateAudio)

Advances the scene by the given time delta, in milliseconds.
  • Throws an Error if elapsed is not provided.

.refresh()

Refreshes this painter's buffers. Invoked automatically when cube counts are changed.

.setMax(max)

Sets the maximum number of cubes in all directions to the given value.

.setXMax(xMax)

Sets the maximum number of cubes in the X direction to the given value.

.setYMax(yMax)

Sets the maximum number of cubes in the Y direction to the given value.

.setZMax(zMax)

Sets the maximum number of cubes in the Z direction to the given value.

.setRotq(rotq)

Sets the rotation time, in degrees.

.switchAudioMode()

Changes the current audio mode.

.paint()

Paints the current number of cubes using this set rotation time.

If an update listener has been given, it will be invoked when painting is complete.

.setUpdateListener(listener, listenerThisArg)

Sets the update listener to the given function and this-object.

.render()

Renders the painted cubes.

.onKeyDown(key)

Handles key-down events. Returns true if the event was handled.

.createAudioNode(audio)

Creates an AudioNode using the given content in this widget's set audio mode.
