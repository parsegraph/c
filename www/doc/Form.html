.addFieldElement(type, name, value, choices)

Adds a named form-field element using the given value to this form, using the given configuration.

Type may be one of the following values:

Name
textarea
label
button
submit
select
checkbox
other typesNo special behavior.

.clone()

Clones this form, creating a new Form with the same form-field definitions.

.load(src)

Loads the values from the given array, in the same order as the defined form-fields, or a name-value map.

.jsonArray()

Returns a list of this form's values.

.jsonObject()

Returns a name-value map of this form's values.

Returns the value of the field specified by the given name.

.clear()

Sets all fields to null.

.getFieldByName(name)

Returns the field info structure for the field specified by the given name.

Prefers non-label fields over label fields with the same name.

Returns null if not field exists for the given name.

.elementFor(fieldName)

Returns the HTML element for the specified form-field.

.eachField(visitor, visitorThisArg)

Invokes the given visitor function and this-object for each form-field in this form.

.addListener(listener, thisArg)

Adds a new listener for change events to this form.

.update(name, value, action)

Notify listeners that the specified field has been updated to the given value.
