Parsegraph Unicode

Unicode functions and the Unicode character database object.

isRTL(s)

Returns true if the given string contains right-to-left characters. This test is a best-effort and not claimed to be complete.

new parsegraph_Unicode()

Creates a new Unicode character database object. The database begins unloaded.

parsegraph_defaultUnicode()

Returns the parsegraph_UNICODE_INSTANCE, creating and loading a new Unicode character database object if necessary.

.get(codeOrLetter)

Returns the Unicode properties for the given Unicode digit code or the first Unicode character of the given string.

.get() requires .load() to have been called and its asynchronous event completed before loaded values are available for use.

.getCursiveMapping(t)

Returns the cursive mapping for the given Unicode digit code. This mapping is an array of semantic code values to an array of [Isolated, Initial, Medial, Final]. Null is used for non-applicable mappings. The same table of arrays is returned for all calls.

.loadFromString(t)

Parses the given string as a Unicode character database file.

.isArabic(letter)

Returns true if the given letter or digit code is considered an Arabic semantic character.

.isMark(letter)

Returns true if the given letter or digit code is considered a Unicode mark category.

.isArabicDiacritic(letter)

Returns true if the given letter or digit code is considered an Arabic semantic character.

.load(dbURL)

Asynchronously loads the Unicode character database at the given location, returning the XMLHttpRequest used to send the request.

.loaded()

Returns true if the Unicode character database has completed loading.

.setOnLoad(onLoad, onLoadThisArg)

Sets the function that will be invoked when loading has completed.

Throws an error if loading has already completed.

For compatibility with prior versions, this.onLoad will also be called in addition to this listener, if that field is manually set.
