Parsegraph pagingbuffer

parsegraph_BufferPage

new parsegraph_BufferPage(pagingBuffer, renderFunc, renderFuncThisArg)

Returns a new buffer page for use with the given PagingBuffer.

The render function and this-object is used to render the page's buffer contents to GL.

.isEmpty()

Returns true if this buffer page contains no buffer, or if its buffers are currently empty.

.appendData(attribIndex, ...)

Adds each of the specified values to the specified buffer of this page.
  • Throws if attribIndex does not specify a real buffer.

.appendData(attribIndex, valueArray)

Adds each of the values of the given array to the specified buffer this page.
  • Throws if attribIndex does not specify a real buffer.

.appendRGB(attribIndex, color)

Appends the RGB values of the given color object to the specified buffer of this page.
  • Throws if attribIndex does not specify a real buffer.

.appendRGBA(attribIndex, color)

Appends the RGBA values of the given color object to the specified buffer of this page.
  • Throws if attribIndex does not specify a real buffer.

parsegraph_PagingBuffer

Manages the low-level paging of vertex attributes.

new parsegraph_PagingBuffer(gl, program)

Creates a new paged buffer using the given GL content and GL program for rendering.

parsegraph_createPagingBuffer(gl, program)

Alias for parsegraph_PagingBuffer's constructor.

.isEmpty()

Returns true if this paging buffer contains no pages, or its pages are empty.

.addPage(renderFunc, renderFuncThisArg)

Creates a new BufferPage and adds it to the end of this paging buffer. The new page is returned.

.getWorkingPage()

Returns the current working page that will have values appended.
  • Throws if this paging buffer has not had any pages added.

.defineAttrib(name, numComponents, drawMode)

Defines an attribute for data entry.

name
the attribute name in this paging buffer's GL program
numComponents
the number of components in the named attribute type (1, 2, 3, or 4)
drawMode
the WebGL draw mode. Defaults to gl.STATIC_DRAW

.appendRGB(...)

Appends the given color RGB components to the current working page.

.appendRGBA(...)

Appends the given color RGBA components to the current working page.

.appendData(...)

Appends the given floating-point arguments to the current working page.

If an array or array-like object is given, it will be iterated to extract its component values.

.clear()

Deletes all buffers and empties values.

.renderPages()

Sets up vertex attribute buffers and calls drawArrays for each page. 
gl.drawArrays(gl.TRIANGLES, 0, numVertices)
where numVertices is calculated from the appended data size / component count. The least-filled buffer is used for the size, if the sizes differ.
