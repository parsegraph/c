Alpha WeetPainter

Draws 3d faces in a solid color.

new alpha_WeetPainter(gl)

Creates a new WeetPainter using the GL context.

.Init(numCubes)

Initializes this painter's buffers for drawing the given number of cubes.

.Cube(m)

Draws a cube centered at the given world matrix.

.Clear()

Resets this painter's buffer pointers.

.Draw(viewMatrix)

Draws the painted cubes using the given view matrix.
