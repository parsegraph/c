Parsegraph SpotlightPainter

Paints translucent spotlights.

new parsegraph_SpotlightPainter(gl, shaders)

Creates a new painter using the given GL context and compiled shaders object.

.drawSpotlight(cx, cy, radius, color)

Draws a circular spotlight centered at the specified position and of the given size and color.

.drawRectSpotlight(cx, cy, w, h, color)

Draws a rectangular spotlight of the specified center, size, and color.

.clear()

Clears this painter's buffers.

.render(world, scale)

Renders this painter's spotlights using the given world matrix and scale.
