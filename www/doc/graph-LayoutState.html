Parsegraph LayoutState

Enumeration of Node layout states.

parsegraph_NULL_LAYOUT_STATE

Constant that refers to a null layout state.

parsegraph_NEEDS_COMMIT

Constant that indicates the Node requires a commit.

parsegraph_COMMITTED_LAYOUT

Constant that indicates the Node has committed its layout.

parsegraph_IN_COMMIT

Constant that indicates the Node is in the process of committing its layout.

parsegraph_nameLayoutState(given)

Returns a string that describes the given layout state.
  • Throws an error if the given layout state constant is unrecognized.
