Alpha Cam

camera is a special case of physical
so special that I've opted to not "descend it"
it is always following a physical
and it passes information to and from physicals

new alpha_Camera(surface)

Creates and returns a new alpha_Camera using the given surface.

Throws an error if surface is not defined.

.toJSON()

Returns a object, described like the following, that describes the current state of this camera:
NameValue
position[x, y, z]
orientation[rx, ry, rz, rw]

.restore(json)

Set this camera's position and orientation from the given object whose format is like that returned by .toJSON().

.SetFovX(fovX)

Sets the field-of-view, in degrees, for this camera using the X-axis.

Setting this value resets the value of fovY.

.SetFovY(fovY)

Sets the field-of-view, in degrees, for this camera using the Y-axis.

Setting this value resets the value of fovX.

.GetFovX()

Gets the field-of-view, in degrees, of this camera.

If fovX has been directly set using .SetFovX(), then this returns that value directly. Otherwise, the field-of-view is computed from fovY.

.GetFovY()

Gets the field-of-view, in degrees, of this camera. Same as fovX, but for fovY.

.SetProperFOV(vpWidth, eyeDistance)

Sets the fov. Unless you have a huge screen and sit very close I do not recommend.

width = width of the viewport
distance = distance of eyes from viewport
use the same units for both;

.SetZoom(factor)

Sets this camera's zoom to the given value assuming factor is >= 1.

Returns the given value, or false if factor < 1.

.GetZoom()

Return this camera's zoom factor.

.SetZoomSpeed(speed)

Sets this camera's zoom speed to the given value.

.ZoomIn(bind, elapsed)

Adjusts this camera's zoom factor, increasing it by zoomSpeed^(bind*elapsed).

bind is clamped to (0, 1]. elapsed is a real value representing the time spent "zooming in"

The computed zoom is clamped such that it will be at least 1.

  • Returns false if bind <= 0; otherwise, the computed zoomFactor is returned.

.ZoomOut(bind, elapsed)

Adjusts this camera's zoom factor, decreasing it by zoomSpeed^(bind*elapsed).

bind is clamped to (0, 1]. elapsed is a real value representing the time spent "zooming in"

The computed zoom is clamped such that it will be at least 1.

  • Returns false if bind <= 0 or if elapsed is false; otherwise, the computed zoomFactor is returned.

.CancelZoom()

Sets this camera's zoom factor to 1.

.ZoomUntil(zoom, bind, elapsed)

continues to zoom until the specified zoom is reached;
broken until I am less tired
  • Returns false if zoom, bind, or elapsed are false, or if bind is non-positive.

.SetFarDistance(distance)

Sets the farDistance to the given value. Anything further than this is clipped.

.GetFarDistance()

Returns this camera's far distance. Anything further than this is clipped.

.SetNearDistance(distance)

Sets the near distance for this camera to the given distance. Anything nearer than this is clipped.

.GetNearDistance()

Returns this camera's near distance. Anything nearer than this is clipped.

.UpdateProjection()

Updates this camera's projection matrix using the camera's surface's container's dimensions.

See also WebGL Fundamentals - Resizing the Canvas.

.UpdateProjection(width, height)

Updates this camera's projection matrix using the specified dimensions.

See also WebGL Fundamentals - Resizing the Canvas.

.SetOrientation(...)

Sets this camera's orientation quaternion using the given arguments.

.GetOrientation()

Returns this camera's orientation, as a quaternion.

.SetRotationSpeeds(x, y)

Sets this camera's rotation speeds, in radians per second, to the given values.

.GetRotationSpeeds()

Directly returns this camera's rotation speeds, as an array of [xSpeed, ySpeed].

.SetRotationSpeed(speed)

Sets both of this camera's rotation speeds to the given value.

.Pitch(angle)

Rotate the camera about the X axis.

.Turn(angle)

Rotate the camera about the Y axis.

.TurnLeft(elapsed)

Turns the camera left by the specified amount of this camera's rotation Y-speed.

.TurnRight(elapsed)

Turns the camera right by the specified amount of this camera's rotation Y-speed.

.PitchUp(elapsed)

Turns the camera up by the specified amount of this camera's rotation X-speed.

.PitchDown(elapsed)

Turns the camera down by the specified amount of this camera's rotation X-speed.

.AlignParentToMy(x, y)

Aligns this camera's parent to this camera's X or Y axis, as determined by the given flags.

.SetPosition(vector)

Calls .SetPosition(vector[0], vector[1], vector[2]).

.SetPosition(x, y, z)

Sets this camera's position to the given value, returning this camera's position.

.SetRange(range)

Sets this camera's position Z value to the given value.

.GetPosition()

Returns this camera's position, as a vector.

.ChangePosition(x, y, z)

Adds the given components to this camera's position.

.SetOffset(vec)

Sets this camera's offset to the given vector. This is the offset from the physical.

.SetOffset(x, y, z)

Sets this camera's offset to the given value. This is the offset from the physical.

.GetOffset()

Returns this camera's offset, as a vector.

.ChangeOffset(vec)

Adds the components of the given vector to this camera's offset.

.ChangeOffset(x, y, z)

Adds the given components to this camera's offset.

.SetMaxRange(maxRange)

Sets the max range of this camera to the given value, returning the new value.

.GetMaxRange()

Returns this camera's max range.

.Warp(distance)

Warps this camera's position by the given distance.

Camera movement is easy; it can only move in and out.

.WarpIn(distance)

Alias for Warp(-distance).

Camera movement is easy; it can only move in and out.

.WarpOut(distance)

Alias for Warp(distance).

Camera movement is easy; it can only move in and out.

.SetSpeed(speed)

Directly sets this camera's speed.

Since we can only move in one direction, there isn't any velocity; these are the commands needed for expected movement.

.GetSpeed()

Returns the speed of this camera.

.MoveForward(elapsed)

Moves the camera forward by the given relative amount, determined by this camera's speed.

.MoveBackward(elapsed)

Moves the camera backward by the given relative amount, determined by this camera's speed.

.IsGoodLineageFor(prospectiveChild)

Returns true, because CAMERAS MAKE THE BEST PARENTS.

.GetInvisiblePhysical(parent)

Creates a new physical positioned and oriented like this camera.

.Disengage()

Enables free floating.

.Engage()

Sends it back to its previous physical

.SetParent(parent)

Sets this camera's parent to the given phyiscal.

Setting the camera to itself sets it to an invisble physical.

.GetParent()

Returns this camera's parent.

.GetModelMatrix()

Combine position, offset and orientation into a matrix.
