parsegraph_buildGlyphAtlas()

Constructs a new GlyphAtlas using the default font, font size, and font color.

parsegraph_clearPerformanceCounters()

Clears the frame's performance counters.

parsegraph_showGraph(rootNode)

Show a basic graph given a parsegraph_Node.

parsegraph_initialize(mathMode)

Initializes the Parsegraph environment. If mathMode is true, then blocks and slots appear grid-like with tight spacing.

parsegraph_NATURAL_GROUP_SIZE

The natural group size that can be used as a limit before nodes should be split into separate PaintGroups.

parsegraph_TOUCH_SENSITIVITY

A multiplier for touch deltas.

parsegraph_MOUSE_SENSITIVITY

A multiplier for mouse deltas.

parsegraph_RIGHT_TO_LEFT

Whether Node's forward and backward are initially switched.

parsegraph_NULL_NODE_DIRECTION

Null node direction.

parsegraph_INWARD

Constant that refers to the direction for Nodes positioned visually inside their parents.

parsegraph_OUTWARD

Constant that refers to the direction for Nodes that encompass their parents.

This direction cannot be spawned.

parsegraph_DOWNWARD

Constant that refers to the direction for Nodes that are beneath their parents.

parsegraph_UPWARD

Constant that refers to the direction for Nodes that are above their parents.

parsegraph_BACKWARD

Constant that refers to the direction for Nodes that are semantically behind of their parents.

parsegraph_FORWARD

Constant that refers to the direction for Nodes that are semantically ahead of their parents.

parsegraph_NUM_DIRECTIONS

Returns the number of supported Node directions.

parsegraph_NULL_AXIS

Constant for a null node axis.

parsegraph_HORIZONTAL_AXIS

Constant that refers to the horizontal Node axis (FORWARD and BACKWARD directions).

parsegraph_VERTICAL_AXIS

Constant that refers to the vertical Node axis (UPWARD and DOWNWARD directions).

parsegraph_NULL_NODE_TYPE

parsegraph_BUD

Constant that refers to the small, circular Node type called 'bud'.

parsegraph_SLOT

Constant that refers to the regular-sized, block-shaped Node type called 'slot'.

parsegraph_BLOCK

Constant that refers to the regular-sized, block-shaped Node type called 'block'.

parsegraph_SLIDER

Constant that refers to the special slider Node type called 'slider'.

parsegraph_SCENE

Constant that refers to the scene-containing Node type called 'scene'.

parsegraph_DEFAULT_NODE_TYPE

Returns the default Node type.

parsegraph_NULL_NODE_FIT

Constant that refers to a null Node fitting.

parsegraph_NODE_FIT_EXACT

Constant that refers to a exact Node fitting.

parsegraph_NODE_FIT_LOOSE

Constant that refers to a loose Node fitting.

parsegraph_MAX_PRESS_RELEASE_DELAY

Returns the maximum delay in milliseconds before the hope for a double-click is abandoned.

parsegraph_BACKGROUND_COLOR

Returns the default background color.

parsegraph_UPSCALED_FONT_SIZE

Returns the size used for drawing fonts to the GlyphAtlas textures.

parsegraph_RENDERED_FONT_SIZE

Returns the size used for rendering fonts in the world.

parsegraph_WRAP_WIDTH

Returns the default width used for word wrapping.

parsegraph_SHRINK_SCALE

The scale at which shrunk nodes are shrunk.

parsegraph_FONT_SIZE

Base font size.

parsegraph_LINE_THICKNESS

The thickness (diameter) of the line.

parsegraph_LINE_COLOR

Returns the color used for drawing lines.

parsegraph_SELECTED_LINE_COLOR

Returns the color used for drawing highlighted lines.

parsegraph_BUD_RADIUS

Returns the size for a bud's radius. Many values are computed from this value.

parsegraph_MIN_BLOCK_HEIGHT

Returns the minimum height for block-shaped Nodes.

parsegraph_MIN_BLOCK_WIDTH

Returns the minimum width for block-shaped Nodes.

parsegraph_HORIZONTAL_SEPARATION_PADDING

Returns the default horizontal padding used to separate Nodes.

parsegraph_VERTICAL_SEPARATION_PADDING

Returns the default vertical padding used to separate Nodes.

parsegraph_BUD_LEAF_SEPARATION

The separation between leaf buds and their parents.

parsegraph_BUD_TO_BUD_VERTICAL_SEPARATION

Return the verticals separation used between buds.

parsegraph_Style

NameDescription
minWidthMinimum rendered Node width, in world units.
minHeightMinimum rendered Node height, in world units.
horizontalPaddingHorizontal space between the content and the border, in world units.
verticalPaddingVertical space between the content and the border, in world units.
borderColorColor of the border.
backgroundColorColor of the background.
selectedBorderColorColor of the border when the Node is selected.
selectedBackgroundColorColor of background when the Node is selected.
brightnessMultiplier of the spotlight used when the Node is selected.
borderRoundnessValue between 0 and 1 that defined how rounded the border is. 1 is circular, 0 is square.
borderThicknessValue between 0 and 1 that defined the amount of the Node that is devoted to the border. Does not affect layout, just appearance.
maxLabelCharsNot used.
fontColorThe color used for Node text.
selectedFontColorThe color used for Node text when the Node is highlighted.
fontSizeThe font size in world units.
letterWidthA multiplier of the font-size used to define the default letter width.
verticalSeparationThe distance in world units between nodes of this style and their vertical child Nodes.
horizontalSeparationSame as verticalSeparation, but for horizontal child Nodes.

The style used for slot node types.

parsegraph_SCENE_STYLE

The style used for scene node types.

parsegraph_copyStyle(type)

Returns a new Style that is a copy of the given style or named style.

parsegraph_style(type)

Returns a style object for the given Node type name.
