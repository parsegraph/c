Parsegraph timing

parsegraph_timeout(name, timeoutMs)

Returns a function that saves when this function was called. When the returned function is invoked, it ensures that the given timeout has not yet been exceeded. If it has, an error that mentions the given name is thrown.

parsegraph_timeout(timeoutMs)

Returns a function that saves when this function was called. When the returned function is invoked, it ensures that the given timeout has not yet been exceeded. If it has, an error is thrown.

parsegraph_timeout()

Returns a function that saves when this function was called. When the returned function is invoked, it ensures that the default timeout has not yet been exceeded. If it has, an error is thrown.

parsegraph AnimationTimer

A one-shot timer implemented using requestAnimationFrame.

new parsegraph_AnimationTimer()

Creates and returns a new, unscheduled timer.

.fire()

Forwards event arguments to this timer's listener.

.schedule()

Schedules the timer for activation on the next animation frame.

This does nothing if the timer is already scheduled.

.setListener(listener, thisArg)

Sets this timer's listener to the given function and this-object.

.cancel()

Cancels the timer's activation if scheduled.

This does nothing if the timer is not scheduled.

parsegraph_TimeoutTimer()

A one-shot timer implemented using window.setTimeout.

new parsegraph_TimeoutTimer()

Creates a new, unscheduled timer with a zero-millisecond delay.

.fire()

Forwards event arguments to this timer's listener.

.setDelay(ms)

Sets the delay, in milliseconds.

.delay()

Gets the delay, in milliseconds.

.setListener(listener, thisArg)

Sets this timer's listener to the given function and this-object.

.cancel()

Cancels the timer's activation if scheduled.

This does nothing if the timer is not scheduled.
