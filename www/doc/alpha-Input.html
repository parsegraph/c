Alpha Input

new alpha_Input(surface, camera)

Creates a new Input object using the given surface and camera.

.Get(key)

Returns whether the specified key is currently pressed.

.SetMouseSensitivityX(sensitivity)

Sets the X-axis mouse sensitivity.

.GetMouseSensitivityX()

Returns the X-axis mouse sensitivity.

.SetMouseSensitivityY(sensitivity)

Sets the Y-axis mouse sensitivity.

.GetMouseSensitivityY()

Returns the Y-axis mouse sensitivity.

.SetMouseSensitivity(sensitivity)

Sets both axis sensitivities to the given value.

.MouseLeft()

Returns the current mouse movement delta in the left direction. Clamped at least zero.

.MouseRight()

Returns the current mouse movement delta in the right direction. Clamped at least zero.

.MouseUp()

Returns the current mouse movement delta in the up direction. Clamped at least zero.

.MouseDown()

Returns the current mouse movement delta in the down direction. Clamped at least zero.

.MouseWheelUp()

Returns the wheel delta in the upwards direction, in ticks.

Mouse wheel data is stored in 1/8 of a degree. This returns how many ticks of a mousewheel of standard resolution has been seen before an .Update().

.MouseWheelDown()

Returns the wheel delta in the downwards direction, in ticks.

.MouseWheelDegreesUp()

Returns the wheel delta in the upwards direction, in degrees.

.MouseWheelDegreesDown()

Returns the wheel delta in the downwards direction, in degrees.

.Update(elapsed)

Updates the scene with the given input state and given elapsed time.

Sets the start to the end, and clears mousewheel totals.
