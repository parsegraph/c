Parsegraph GlyphAtlas

new parsegraph_GlyphPage()

Returns a new, empty glyph page.

new parsegraph_GlyphAtlas(fontSizePixels, fontName, fillStyle)

Creates a new glyph atlas using the given font.

.setUnicode(uni)

Sets the Unicode character database to the given object.
  • Throws an Error if the given database has not completed loading.

.unicode()

Return this glyph atlas's Unicode character database.

.toString()

Returns a developer-friendly string naming this glyph atlas.

.getGlyph(glyph)

Returns the glyph data for the given string or character code.
NameDescription
letterglyph
xThe left-most X coordinate of this glyph in its glyph page.
yThe top-most Y coordinate of this glyph in its glyph page.
widthThe computed width of this glyph.
heightThe computed height of this glyph.
glyphPageThe page that contains this glyph.

.get(glyph)

Alias for .getGlyph(glyph).

.hasGlyph(glyph)

Returns true if there is glyph data for the named glyph.

.has(glyph)

Alias for .hasGlyph(glyph).

.update(gl)

Updates the given WebGL instance with this texture. 

// Updates the standard GL instance.
ga.update();
// Updates the given GL instance and clears old one.
ga.update(gl);

.clear()

Deletes all loaded GL textures.

.imageData()

Returns binary image data that describes the current page.

.getImageData()

Alias for .imageData().

.needsUpdate()

Returns true if this glyph atlas is flagged for update.

.restoreProperties()

Sets the context's font and fill-style.

.font()

Returns a CSS string describing the font in use.

.canvas()

Returns the canvas used for rendering the 2D glyph page.

.maxTextureWidth()

Returns the maximum texture width, in pixels.

.letterHeight()

Returns the height used for all letters.

.fontBaseline()

Returns the baseline at which the font is rendered.

.fontSize()

Directly returns the set font size.

.fontName()

Directly returns the set font name.

.isNewline(c)

Returns true if the given string is a newline character.
