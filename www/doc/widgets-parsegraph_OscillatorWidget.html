parsegraph_OscillatorWidget

Widget to control an OscillatorWidget audio node.

new parsegraph_OscillatorWidget(graph)

Creates a new widget using the given Graph. The audio node is not created until .build() is called.

.build(audio)

Initializes this widget by creating a new OscillatorNode using the given AudioContext.

The OscillatorNode is returned.

.setOscillatorType(oscType)

Directly sets this widget's oscillator type.

.setOscillatorFrequency(value)

Directly sets this widget's oscillator frequency value.

.setOscillatorDetune(value)

Directly sets this widget's oscillator detune value.

.refreshTypes()

Refreshes the widget's state using the set values.

.node()

Returns the Node that contains this widget. The Node is created if necessary.
