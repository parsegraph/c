Parsegraph Camera

The camera controls input from the user on a surface to manage visualization of a 2D orthographic world.

parsegraph_CLICK_DELAY_MILLIS

The amount of delay between clicks, in milliseconds, that will be allowed before the hope for a double-click is abandoned.

new parsegraph_Camera(surface)

Creates a new parsegraph_Camera using the given surface, positioned at the origin, with a scale of 1.

An error is thrown if no surface is provided.

parsegraph_containsAll(viewportX, viewportY, viewWidth, viewHeight, cx, cy, width, height)

Returns whether the specified viewport contains all of the specified rect.

parsegraph_containsAny(viewportX, viewportY, viewWidth, viewHeight, cx, cy, width, height)

Returns whether the specified viewport contains any of the specified rect.

.zoomToPoint(scaleFactor, x, y)

Zooms to the given point.

.setOrigin(x, y)

Sets the camera's origin to the specified position.

.toJSON()

Returns a new JSON that can be passed to JSON.stringify() and used with .restore() to recreate this camera's current state.

.restore(json)

Changes this camera's state to the given JSON object, as defined by .toJSON().

.surface()

Returns the surface used by this camera.

.graph()

Same as .surface(). Used for backwards compat.

.scale()

Returns this camera's current scale.

.x()

Returns this camera's current position along the X-axis.

.y()

Returns this camera's current position along the Y-axis.

.setScale(scale)

Sets this camera's scale to the given value.

.toString()

Returns a programmer-friendly string representation of this camera.

.adjustOrigin(x, y)

Adjusts this camera's origin by the given delta.

.worldMatrix()

Computes and returns a new world matrix that represents this camera's current position.

.aspectRatio()

Returns this camera's aspect ratio, as computed during project().

.width()

Returns this camera's viewport width, as computed during project().

.height()

Returns this camera's viewport height, as computed during project().

.canProject()

Returns whether this camera's surface has usable dimensions.

.project()

Returns a new projection matrix for this camera and sets the computed viewport dimensions and aspect ratio.

Throws an error if canProject() returns false.
