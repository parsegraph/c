parsegraph_TestResult

Status is either "Successful", "Failed", or "Crashed"

Result is undefined for "Successful", anything else for "Failed", and the exception thrown for "Crashed"

The originating test is also available.

new parsegraph_TestResult(testStatus, testResult, test)

Creates a new TestResult with the given values.

.testStatus()

Retrieves this result's given status.

.testResult()

Retrieves this result's given result.

.test()

Retrieves this result's given test object.

parsegraph_TestSuite

A collection of tests, able to be run and recorded.

A TestSuite called parsegraph_AllTests is created that contains all created TestSuites, because they are added in the constructor. dontAutoadd disables this behavior for the created TestSuite.

new parsegraph_TestSuite(name, dontAutoadd)

Creates and returns a new TestSuite with the given name.

If dontAutoadd is false and parsegraph_AllTests is defined, then this test is added as a test of that TestSuite.

parsegraph_AllTests

The TestSuite that conventionally contains all other test suites.

.addTest(testName, runner, runnerThisArg)

Appends the specified test to this test suite.

.run(listener, listenerThisArg)

parsegraph_TestSuiteResult

new parsegraph_TestSuiteResult()

Creates and returns a new test suite result.

.connect(container)

Connects this test suite result to the given HTML element.

The result is disconnected if necessary.

.disconnect()

Removes this result's content from its container.

.resultList()

Returns this result's result list HTML element.

.aggregateResult()

Returns this result's aggregate result HTML element.

.testStarted()

Increment the started-tests counter.

.testsStarted()

Return the number of tests that have been started.

.testsSuccessful()

Return the number of tests that have succeeded.

.testsFailed()

Return the number of tests that have failed with a error result.

.testsCrashed()

Return the number of tests that have failed by throwing exceptions.

.testsFinished()

Return the number of tests that have succeeded or failed.

.testFinished(result)

Mark that a test has finished with the result of "Successful", "Failed", or "Crashed".

.isSuccessful()

Returns true if this result has completed all of its tests and did not record any failures.

.toString()

Returns a user-friendly string describing this result.
