Parsegraph CameraBoxPainter

Painter used to render camera boxes using the given GL context.

new parsegraph_CameraBoxPainter(gl, glyphAtlas, shaders)

Creates a new painter using the given GL rendering arguments.

.clear()

Clears this painter's buffers.

.drawBox(name, rect, scale)

Draws a named camera box of the given dimensions and scale.

.render(world)

Renders the painted camera boxes using the given matrix.
