alpha_Color

A simple class to make it easier to create RGB colors. Color values are in the range [0, 1].

This class's component values may be used like an array.

new alpha_Color(...)

Creates a new color. If arguments are given, they are passed to .Set().

.asRGB()

Returns a CSS-friendly string describing this color.

.Set(other)

Sets this color's value to the given color.

.Set(r, g, b)

Sets this color's value to the given color values.

.Set(v)

Sets this color's values to the given greyscale value.

.Set(hexString)

Sets this color's values to the color described by the given string, like '#ff00ff', or 'aabbcc'.

.Equals(other)

Returns true if this color's values are equal to the values of the given color.

.Equals(r, g, b)

Returns true if this color's values are equal to the given color values.

.Equals(v)

Returns true if all of this color's values are equal to the given color value.

.toString()

Returns a string of the printf(1) format "{%f, %f, %f}".

alpha_Skin()

The skin object is simply an ordered list of colors. There is one color element for each vertex of each face of a shape.

A skin can only be applied to a shape with the same number of vertices.

Skins are immutable once created.

new alpha_Skin()

Creates an immutable, empty Skin.

new alpha_Skin(facesArray)

Creates an immutable Skin using the given array of faces of the following form: 
// green, brown are alpha_Colors.
var skin = new alpha_Skin([
    [green, green, green, green], // face 1 has 4 vertices
    [brown, brown, brown, brown], // face 2
    [brown, brown, brown, brown] // face 3
]);

new alpha_Skin(...)

Creates an immutable Skin using the given arguments as the array of faces. 
var skin = new alpha_Skin(
    [lbrown, lbrown, lbrown, lbrown], // top
    [lbrown, lbrown, dbrown, dbrown], // front
    [lbrown, lbrown, dbrown, dbrown], // left
    [lbrown, lbrown, dbrown, dbrown], // back
    [lbrown, lbrown, dbrown, dbrown], // right
    [dbrown, dbrown, dbrown, dbrown] // bottom
);

.forEach(callback, thisArg)

Invokes callback(thisArg, color, colorIndex, this) for each color of this skin.

alpha_Face

Face is a simple grouping of vertices designed to be rendered by 1 call of GL_QUADS or its ilk. 
local cubeTop = new alpha_Face(alpha_QUADS, vector, vector, vector, vector);
Face does not copy the vectors.

Once it is passed to a shape the shape will copy it, so do not reuse until after the face is applied to a shape.

alpha_TRIANGLES

Specifies that the face's vertices are to be interpreted as a list of triangles.

alpha_QUADS

Specifies that the face's vertices are to be interpreted as a list of quads.

new alpha_Face(drawType, vertexArray)

Creates a new face of the given draw type (either alpha_TRIANGLES or alpha_QUADS) and the given array of alpha_Vectors.

new alpha_Face(drawType, ...)

Creates a new face of the given draw type (either alpha_TRIANGLES or alpha_QUADS) and a variable argument list of alpha_Vectors.

.Clone()

Creates and returns a new face that is a deep copy of this face.

.toString()

Returns a string describing each color of this face.

alpha_Shape

Shape is a list of faces.

new alpha_Shape(...)

Constructs a new shape using the given list of faces.

alpha_BlockTypes

Blocktype stores named combinations of Shape(pos vec) + Skin(color vec). 
var stone = new alpha_BlockType("stone", "cube", Stone, graySkin)
BlockType automatically loads created BlockTypes into the BlockIDs table
it is some sort of hybrid object / masterlist

new alpha_BlockTypes()

Constructs a new BlockTypes store.

.Load(descSkin, descShape, skin, shape)

Alias for .Create().

.Create(descSkin, descShape, skin, shape)

Creates a blocktype and returns the id.
  • Throws if the name is already in use, or if the skin's size does not match the shape's size.

.Get(id)

Returns the [shape, skin] blockType for the given id.

.Get(descSkin, descShape)

Returns the [shape, skin] blockType with the given name.

new alpha_Block(id, x, y, z, orientation)

Creates a new alpha_Block of the given block type at the given position. The orientation is one of [0, 24) possible directions.

new alpha_Block(id, vec, orientation)

Creates a new alpha_Block of the given block type at the specified position. The orientation is one of [0, 24) possible directions.

alpha_createBlock(...)

Alias for alpha_Block's constructor.

.Equals(other)

Returns true if this block's position is equal to the given block's position.

.GetAngleAxis()

Returns a [vec, angle] array that represents this block's current orientation.

.GetQuaternion(actual)

Returns the quaternion that represents this block's orientation.

If actual is true, a new quaternion is created; otherwise, the orientation is directly returned.
