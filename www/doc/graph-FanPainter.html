new parsegraph_FanPainter(gl)

Creates a new painter of radius 250 using the given GL context.
  • Throws an Error if a GL context is not given.

.selectDeg(userX, userY, startAngle, spanAngle, startColor, endColor)

Draws a selection at the given world coordinates highlighting the given angle with the given Colors.

.selectRad(userX, userY, startAngle, spanAngle, startColor, endColor)

Draws a selection at the given world coordinates highlighting the given angle with the given Colors.

.setAscendingRadiusfunction(ascendingRadius)

Sets the distance from the center to the brightest point.

.setDescendingRadius(descendingRadius)

Sets the distance from the brightest point, to the invisible outer edge.

.setSelectionAngle(selectionAngle)

Sets the selection angle in radians from [-Math.PI, Math.PI], which is an area of radial brightness.

If null is given, no selection highlight is drawn. This is the default.

.setSelectionSize(selectionSize)

Sets the selection angle in radians from [0, 2*Math.PI], which is the size of the selected highlight.

If null is given, no selection highlight is drawn. This is the default.

.clear()

Clears all buffers.

.render(viewMatrix)

Renders all painted fans using the given matrix.
