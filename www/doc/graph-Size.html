Parsegraph Size

Size object. This object can be used like an array.

new parsegraph_Size()

Creates a new Size of zero size.

new parsegraph_Size(width, height)

Creates a new Size of the specified size.

parsegraph_createSize(width, height)

Alias for parsegraph_Size's constructor.

.clear()

Resets this Size's width and height to zero.

.reset()

Alias for .clear().

.scale(factor)

Scales this Size's components by the given factor.

.scaled(factor)

Returns a new Size scaled by the given factor.

.width()

Returns the width and the first component of this Size.

.setWidth(width)

Directly sets the width and the first component of this Size.

.height()

Returns the height and the second component of this Size.

.setHeight(height)

Directly sets the height and the second component of this Size.

.toString()

Returns a developer-friendly string describing the value of this Size.
