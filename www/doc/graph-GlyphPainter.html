Parsegraph GlyphPainter

Draws 3d faces in a solid color.

new parsegraph_GlyphPainter(gl, glyphAtlas, shaders)

Creates a new painter using the rendering arguments.

The GL context is used immediately to create the shader program.

  • Throws an Error if glyphAtlas is null.

.setColor(color)

Sets the current font color to the given Color value.

.setColor(...)

Sets the current font color to the specified Color value.

.setBackgroundColor(color)

Sets the current font highlight color to the specified Color value.

.setBackgroundColor(...)

Sets the current font highlight color to the specified Color value.

.fontSize()

Returns this painter's glyph atlas's current font size.

.glyphAtlas()

Returns this painter's glyph atlas.

.drawGlyph(glyphData, x, y, fontScale)

Draws the specified glyph at the given world coordinates and scale.

.clear()

Clears all buffers.

.render(world, scale)

Renders all drawn glyphs using the specified matrix and scale.
