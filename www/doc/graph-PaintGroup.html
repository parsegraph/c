Parsegraph PaintGroup

new parsegraph_PaintGroup(root, x, y, userScale)

Creates a new PaintGroup using the given root node with the given scale and center.

new parsegraph_PaintGroup(root)

Creates a new PaintGroup using the given root node with a scale of 1.0 centered at the origin.

.clear()

Removes all child PaintGroups from this paint group.

.setOrigin(x, y)

Changes this PaintGroup's origin to the specified position.
  • Throws if either coordinate is NaN.

.scale()

Returns the current scale.

.setScale(scale)

Changes this PaintGroup's scale to the specified value.
  • Throws if the given value is NaN.

.root()

Returns the given root Node of this PaintGroup.

.nodeUnderCoords(x, y)

Returns the node under the given world coordinates.

.assignParent(parentGroup)

Sets this PaintGroup's parent to the given value.

.markDirty()

Marks this PaintGroup as needing to be repainted.

.isDirty()

Returns true if this PaintGroup has been marked as dirty, or has been just constructed and never painted.

.painter()

Returns the NodePainter used to paint this PaintGroup.

.isEnabled()

Returns true if this PaintGroup is enabled. Disabled PaintGroups are not painted, and their descendents are merged into the parent's PaintGroup.

.enable()

Enables this PaintGroup.

.disable()

Disables this PaintGroup.

.paint(gl, backgroundColor, glyphAtlas, shaders, timeout)

Paints all the nodes in this paint group. paint should be called until it returns true.

.toString()

Returns a developer-friendly string naming this PaintGroup.

.renderIteratively(world, camera)

Renders the painted nodes of this group and all its descendents, using the given world matrix and camera.

.traverseBreadth(callback, callbackThisArg)

Calls the given function and this-object for this paint group and all its descendents, in breadth-first order.

.render(world, camera)

Renders the painted nodes of this group, using the given world matrix and camera.

Does nothing if the PaintGroup is disabled, or if it has not been painted.

parsegraph_foreachPaintGroupNodes(root, callback, callbackThisArg)

Returns a painter's algorithm-friendly list of nodes that use the same paint group as the given node. The given node is included.

parsegraph_findChildPaintGroups(root, callback, callbackThisArg)

Traverses the paint groups in the given node in a painter's algorithm-friendly manner.

This method does not consider whether the paint group is enabled or disabled.
