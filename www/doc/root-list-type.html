List types

Each environment has a root list ID. This list ID references a list_item. All list_items have type and value fields, but the list_items referenced by environment are assumed to have globally unique type values - child list items can have type-specific child type values.

The type is used to identify which handler parsegraph will use to create its response.

.show(mediaType)

Generates a response that visualizes this list item in the given media type.

.clone()

Returns a new list item that is a deep clone of this list item.
