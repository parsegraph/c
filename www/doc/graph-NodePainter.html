new parsegraph_NodePainter(gl, glyphAtlas, shaders)

Constructs a new painter using the given GL context, glyph atlas, and prepared shaders.

.bounds()

Returns a rect in world-space that covers this painter's painted area.

.gl()

Returns the GL context used by this painter.

.glyphPainter()

Returns the glyph painter used by this painter.

.setBackground(color)

Directly sets the background color to the given color.

.setBackground(r, g, b)

Sets the background color to the given opaque color.

.setBackground(r, g, b, a)

Sets the background color to the given color values.

.backgroundColor()

Retrieves the current background color.

.render(world, scale)

Renders the painted scene graph.

.enableExtentRendering()

Enables extent painting and rendering.

.disableExtentRendering()

Disables extent painting and rendering.

.isExtentRenderingEnabled()

Returns true if extents are being painted.

.enableBlockRendering()

Enables block and line painting and rendering.

.disableBlockRendering()

Disables block and line painting and rendering.

.isBlockRenderingEnabled()

Returns true if block and line painting and rendering is enabled.

.enableTextRendering()

Enables text painting and rendering.

.disableTextRendering()

Disables text painting and rendering.

.isTextRenderingEnabled()

Returns true if text rendering is enabled.

.enableSceneRendering()

Enables scene painting and rendering.

.disableSceneRendering()

Disables scene painting and rendering.

.isSceneRenderingEnabled()

Returns true if scene painting and rendering is enabled.

.clear()

Clears all painters and textures.

.drawSlider(node, worldX, worldY, userScale)

Draws a slider specified using the given node at the given world coordinates and scale.

.drawScene(node, worldX, worldY, userScale, shaders)

Draws a scene specified using the given node at the given world coordinates and scale.

The given shaders object is used to cache program compilation.

.initBlockBuffer(counts)

Initialize painter buffers using the specified block and extent counts.
NameValue
numBlocksThe number of painted blocks
numExtentsThe number of painted extents.

.countNode(node, counts)

Counts the number of paintable blocks and extents in the given node, adding that total to the given counts object.

See .initBlockBuffer(counts).

.drawNode(node, shaders)

Draws a single node, and the lines extending from it.

.paintLines(node, worldX, worldY, userScale)

Paints the given node's lines extending to its neighbors.

.paintExtent(node, worldX, worldY, userScale)

Paints the given node's extents.

.paintBlock(node, worldX, worldY, userScale)

Paints the block that comprises the given node.
