Parsegraph NodeType

Node type functions. NodeType constants are defined in graph/initialize.js.

parsegraph_nameNodeType(given)

Returns a programmer-friendly string naming the given NodeType.
  • Throws a parsegraph_BAD_NODE_TYPE Error if the given value is unrecognized.

parsegraph_readNodeType(given)

Returns a NodeType constant for the parsed string value. Supports short-hand versions like 'b' for blocks.
